## Getting Started

### Prerequisites
- A modern web browser
- Basic knowledge of HTML/CSS/JavaScript

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/gnanesh2310/1st-website.git
   ```
2. Open `index.html` in your browser

## Usage
Simply open the HTML file in any modern web browser to view the website.
